12 ATTACK TYPES IN CIC-DDoS2019
================================================================================

Network Layer DDoS:
  1. Syn.parquet           - SYN flooding
  2. UDPLag.parquet        - UDP lag attacks
  3. DrDoS_UDP.parquet     - UDP flooding

Amplification Attacks (DrDoS = Distributed Reflection DoS):
  4. DrDoS_DNS.parquet     - DNS amplification
  5. DrDoS_NTP.parquet     - NTP amplification
  6. DrDoS_SNMP.parquet    - SNMP amplification
  7. DrDoS_SSDP.parquet    - SSDP amplification
  8. DrDoS_LDAP.parquet    - LDAP amplification
  9. DrDoS_MSSQL.parquet   - MSSQL amplification
  10. DrDoS_NetBIOS.parquet - NetBIOS amplification

Application Layer DDoS:
  11. WebDDoS.parquet      - Web application DDoS
  12. TFTP.parquet         - TFTP attacks

Each file: ~3-5GB, ~500k-2M network flows, 88 features
